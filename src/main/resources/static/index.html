操作介绍
1、系统分为管理员和医生，医生账户可以由管理员创建得到。
2、管理员和医生都可以上传抑郁症视频，视频上传成功后自动生成未完成的标记任务。
3、医生登录系统，领取任务，点击未完成的任务跳转到标记页面进行标记。
4、医生选取矩形区域单击矩形区域进行放大，在放大后的视频上可以使用画笔进行标注，同时选择对应的标签进行提交。
